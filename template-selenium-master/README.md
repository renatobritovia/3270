# Litonleaning_Automation

Projeto para testes funcionais usando cucumber, JUnit, selenium e testes multiBrowser


#Execucao
Pelo nome da classe:
```maven
mvn clean test -Dtest=ExampleTest
```

Pelo nome da suite:
```maven
mvn clean test -PsuiteFuncional -DtestSuite=testng
```

Todos os testes:
```maven
mvn clean test
```
